# Hindi-Spellchecker

Steps

1. Download the code 
1. Create a virtual env 
1. Activate the virtual env
1. Run requirements.txt file
pip install -r requirements.txt
1. run flask 

One has to choose the language needed later enter the text and click on submit which will display the result.
The english spell correction is done using Spello library for which the following repository was used and integrated in our application 
download the pickle file from this repository
https://github.com/hellohaptik/spello
